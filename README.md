# Spotify Streaming Behavior Analysis (SQL Case Study)

This project analyzes personal Spotify streaming history using SQL to uncover
listening habits, behavioral patterns, and engagement insights.

## Dataset Overview
- Total Records Analyzed: 149,860
- Data Source: Spotify Streaming History
- Key Attributes: Tracks, Artists, Skips, Shuffle Mode, Timestamps

## Key Analysis Performed
- Most-listened artists and tracks
- Skip behavior analysis
- Shuffle usage patterns
- Time-based listening trends
- Monthly streaming activity

## Tools Used
SQL • MySQL • Data Analysis
